# Overview
Code in R that allows researchers to quantify the similarity between two transcribed and annotated texts, considering differences in transcripts with respect to word choice, word order, utterance segmentation, and CLAN annotation marks (i.e., repetitions, revisions, fillers, fragmented speech, paraphasias and onomatopoeas).
